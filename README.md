<h1 align="center">🔧 FMTool</h1>

<p align="center"><b>FMTool</b> — мощный инструмент для проверки Minecraft модов на наличие чита, анализа подозрительных файлов и форензики цифровых следов. Разработан специально для серверов <b>FunMoon</b>.</p>

---

## 🚀 Возможности

### 🔍 Mod Checker
Сканирует JAR-файлы на читы в реальном времени — FMTool проверяет их по <b>300+ сигнатурам</b>, вычисляет SHA-256 хэш, анализирует байт-код и выставляет оценку от <b>0 до 100+</b> с подробным описанием подозрительных строк.

### 📦 Форензика файлов
Не только JAR! Программа умеет анализировать <b>любые файлы</b>:
- <b>PE-анализатор</b> — проверка exe/dll на вирусы
- <b>Энтропия</b> — вычисление случайности данных
- <b>YARA-правила</b> — кастомные проверки
- <b>Строки</b> — извлечение текста

На выходе — файл с <b>5 уровнями угрозы</b> и подробным описанием.

### 📊 Report Window
Два окна с удобной навигацией. Можно сохранить отчёт в виде обычного текста, подсвеченного HTML или структурированного <b>TXT</b> или <b>JSON</b>.

### 🛠️ Встроенные утилиты (14 инструментов)
Панель быстрого доступа к полезным программам для расследования:

| Инструмент | Назначение |
|------------|-----------|
| <b>HxD</b> | Шестнадцатеричный редактор |
| <b>Everything</b> | Мгновенный поиск файлов |
| <b>SystemInformer</b> | Мониторинг процессов |
| <b>Luyten</b> | Декомпилятор Java |
| <b>PrefetchView++</b> | Анализ Prefetch файлов |
| <b>USBDriveLog</b> | Журнал USB устройств |
| <b>JournalTrace</b> | Анализ USN Journal |
| <b>ShellBag Analyzer</b> | Разбор ShellBag |
| <b>ExecutedProgramsList</b> | Список запущенных программ |
| <b>CleaningDetector</b> | Детект чисток |
| <b>PathsParser</b> | Парсинг путей |
| <b>InjGen</b> | Обнаружение инъекций |
| и другие... | |

## 📥 Установка

### Способ 1: Portable версия
1. Скачай FMTool-1.0.2.exe
2. Запусти — без установки

### Способ 2: Установщик
1. Скачай FMTool-Setup-1.0.2.exe
2. Запусти, следуй инструкциям
3. Приложение появится в меню и на рабочем столе

> ⚠️ Антивирус может ругаться — это норма для инструментов безопасности. Добавьте в исключения, если нужно.
