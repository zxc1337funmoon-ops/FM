# 🚀 FMTool

**FMTool** — это удобный инструмент для проверки Minecraft модов на наличие читов, анализа подозрительных файлов и быстрого запуска полезных утилит. Создан специально для сообщества сервера **FunMoon**.

---

## ✨ Возможности

### 🔍 Mod Checker
Перетащи JAR-файл в окно программы — и FMTool проверит его по **300+ ключевым словам**, покажет SHA-256 хеш, выставит оценку угрозы от **0 до 100+** и подсветит самые подозрительные места.

### ⚡ Анализатор файлов
Не только JAR! Программа умеет сканировать **любые файлы**:
- **PE-анализ** — проверка exe/dll структур
- **Энтропия** — выявление упаковщиков и шифровальщиков
- **YARA-правила** — сигнатурный поиск
- **Эвристика** — поведенческий анализ

На выходе — один из **5 уровней угрозы** с подробным описанием.

### 📄 Report Window
Все результаты собираются в удобную таблицу. Можно сортировать по любому столбцу, фильтровать записи и экспортировать отчёт в **TXT** или **JSON**.

### 🧰 Набор утилит (14 инструментов)
Быстрый доступ к полезным программам без лишних движений:

| Утилита | Назначение |
|---------|------------|
| **HxD** | Шестнадцатеричный редактор |
| **Everything** | Мгновенный поиск файлов |
| **SystemInformer** | Продвинутый диспетчер задач |
| **Luyten** | Декомпилятор Java |
| **PrefetchView++** | Анализ Prefetch файлов |
| **USBDriveLog** | Журнал USB устройств |
| **JournalTrace** | Трассировка USN Journal |
| **ShellBag Analyzer** | Анализ ShellBag |
| **ExecutedProgramsList** | Список запущенных программ |
| **CleaningDetector** | Детектор чисток |
| **PathsParser** | Парсер путей |
| **InjGen** | Генератор инжектов |
| и другие | |

### 🌐 Два языка
Переключение между **русским** и **английским** прямо в программе — без перезапуска, без переустановки.

### 🛠 Настройки
- Включение/отключение автозапуска сканирования
- Drag-and-drop (включён по умолчанию)

- Синхронизация темы с системой
- Управление списком модов

## 📥 Установка

### Вариант 1: Портативная версия
1. Скачай `FMTool-1.0.2.exe`
2. Запусти — и пользуйся

### Вариант 2: Установщик
1. Скачай `FMTool-Setup-1.0.2.exe`
2. Запусти, следуй инструкциям
3. Ярлык появится в Пуске и на рабочем столе

> ⚠️ Антивирус может ругаться — это норма для программ такого рода. Добавь в исключения, если доверяешь.
