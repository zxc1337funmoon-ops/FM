# FMTool

**FMTool** — инструмент для проверки файлов на читы, анализа Minecraft модов и поиска подозрительных файлов на ПК. Создан для сервера **FunMoon**.

## Возможности

### 🔍 Mod Checker
Параллельное сканирование JAR-файлов, SHA-256 кэш, 150+ детектов читов, анализ обфускации, взвешенный скоринг (0–100+).

### 📂 Scanner
Сканирование файловой системы: PE-анализ, энтропия, YARA-сигнатуры, белый список библиотек, 5 уровней подозрительности.

### 📄 Report Window
Детальный отчёт с цветовой маркировкой, 4 вкладки (Детект/Классы/Структура/Метаданные), экспорт в TXT/JSON.

### 🚀 Встроенные утилиты
HxD, Everything, SystemInformer, Luyten, PrefetchView++, USBDriveLog, JournalTrace, ShellBag Analyzer, ExecutedProgramsList, CleaningDetector, PathsParser, InjGen, Recaf.

### ⚙️ Системная интеграция
Контекстное меню, ассоциация файлов, авто-обновление, автозапуск, системный трей.

### 🎨 Интерфейс
7 тем (Dark, Arctic, Emerald, Rose, Cherry, Gold, Violet), стеклянный футер, плавные переходы, закруглённые углы, EN/RU локализация.
