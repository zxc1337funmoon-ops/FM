## 🚀 Возможности

### 🔍 Mod Checker

Сканирует JAR-файлы на читы в реальном времени:

- ✅ **300+ сигнатур** — обширная база известных клиентов
- ✅ **SHA-256 хэширование** — уникальный отпечаток каждого файла
- ✅ **Байт-код анализ** — поиск подозрительных паттернов
- ✅ **Оценка угрозы** — от 0 до 100+ с детальным описанием

### 📦 Форензика файлов

Анализирует **любые файлы**, не только JAR:

| Анализатор | Описание |
|------------|----------|
| **PE-анализатор** | Проверка exe/dll на вирусы и вредоносный код |
| **Энтропия** | Вычисление случайности данных для выявления упаковщиков |
| **YARA-правила** | Кастомные сигнатурные проверки |
| **Строки** | Извлечение текстовых строк из бинарных файлов |

На выходе — файл с **5 уровнями угрозы** и подробным отчётом.

### 📊 Report Window

- Два окна с удобной навигацией
- Сохранение отчётов в **TXT**, **JSON** или **HTML** с подсветкой
- Экспорт результатов сканирования одним кликом

## ❓ FAQ

**В: Антивирус удаляет файл. Что делать?**

О: Добавь папку с FMTool в исключения антивируса. Инструменты безопасности часто используют те же приёмы, что и вредоносное ПО — это нормально.

**В: Не открывается Luyten / HxD / другие утилиты?**

О: Убедись, что утилиты установлены. FMTool только открывает их — сами программы нужно скачать отдельно.

**В: Программа не видит Java?**

О: Нажми кнопку «Java JRE 8» в панели утилит — установщик скачает и настроит Java.

**В: Будет ли обновление для macOS / Linux?**

О: Пока нет. FMTool завязан на Windows API (анализ PE, системные утилиты, реестр).
