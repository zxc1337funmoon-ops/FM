# FMTool v1.0 — Полный промпт для AI (генерация C++/Qt проекта)

## Инструкция для AI

Твоя задача — **сгенерировать полный C++20/Qt 6.8+ проект FMTool** по этой спецификации. Все файлы должны быть готовы к компиляции `cmake -B build -G Ninja && cmake --build build`.

**Правила генерации:**
- Все файлы: UTF-8 без BOM, окончания строк CRLF (Windows)
- Русский текст в коде: `QStringLiteral("...")` или `u"..."` (C++20 char8_t)
- Никаких внешних зависимостей кроме Qt 6.8+ Widgets/Core/Gui/Svg + OpenMP + SQLite (QSqlite, встроен в Qt)
- Никаких серверов, API, интернета, телеметрии, регистрации, аккаунтов, истории сканов, статистики
- Все потоки отменяемы (cancel/stop через QAtomicInt или QThread::requestInterruption)
- SHA-256 вычисляется только для файлов, которых нет в SQLite-кеше
- QSS стилизация везде (не QPalette)
- Все окна: QSS border-radius + QGraphicsDropShadowEffect
- Иконки из Qt-ресурсов (resources.qrc)
- **7 тем** (dark, arctic, emerald, rose, cherry, gold, violet) — цвета в секции 12
- Фон окна: сплошной цвет WindowBackground + радиальный glow accent + сетка (белые линии alpha 8, шаг 80px)
- Карточки: линейный градиент lighter→darker, border 1px CardBorder, border-radius 10px

**Порядок генерации (рекомендуемый):**
1. `CMakeLists.txt` (корень проекта)
2. `resources/resources.qrc` + `resources/lang/en.json`, `ru.json` + 10 QSS-тем в `resources/themes/`
3. `src/localemanager.h/.cpp` — менеджер локализации (загрузка JSON, смена языка, сигнал)
4. `src/thememanager.h/.cpp` — менеджер тем (загрузка QSS, смена, fade-анимация)
5. `src/cacherepository.h/.cpp` — SQLite кеш + whitelist (потокобезопасный через QMutex)
6. `src/fileinspector.h/.cpp` — диалог File Inspector (PE-заголовки, SHA-256, метаданные)
7. `src/holycheck.h/.cpp` — анализ .jar (ZIP-распаковка, 10 категорий, malware scan, классификация)
8. `src/minecraftchecker.h/.cpp` — проверка .minecraft (моды, версии, конфиги, загрузчики)
9. `src/programs.h/.cpp` — APPS-менеджер (карточки утилит, извлечение, запуск, PID-трекинг)
10. `src/patterns.h/.cpp` — 6 паттернов Everything с подсветкой синтаксиса
11. `src/logs.h/.cpp` — PowerShell History, CSRSS Strings, HxD Strings с подсветкой сигнатур
12. `src/scanner.h/.cpp` — ядро сканера (фильтры, обход, OpenMP, классификация, Quick Peek, сортировка)
13. `src/mainwindow.h/.cpp` — главное окно (TitleBar, Sidebar, StatusBar, Toast, 6 страниц, tray)
14. `src/main.cpp` — точка входа (QApplication, загрузка темы/языка/кеша, MainWindow)

## 3. CMakeLists.txt (полный)

```cmake
cmake_minimum_required(VERSION 3.16)
project(FMTool VERSION 1.0 LANGUAGES CXX)

set(CMAKE_CXX_STANDARD 20)
set(CMAKE_CXX_STANDARD_REQUIRED ON)
set(CMAKE_AUTOMOC ON)
set(CMAKE_AUTORCC ON)
set(CMAKE_AUTOUIC ON)

find_package(Qt6 REQUIRED COMPONENTS Widgets Core Gui Svg)
find_package(OpenMP QUIET)

if(OpenMP_CXX_FOUND)
    message(STATUS "OpenMP found: ${OpenMP_CXX_FLAGS}")
    set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} ${OpenMP_CXX_FLAGS}")
else()
    message(WARNING "OpenMP not found — compiling without parallel scan support")
endif()

set(SOURCES
    src/main.cpp
    src/mainwindow.cpp
    src/scanner.cpp
    src/holycheck.cpp
    src/minecraftchecker.cpp
    src/programs.cpp
    src/patterns.cpp
    src/logs.cpp
    src/cacherepository.cpp
    src/fileinspector.cpp
    src/localemanager.cpp
    src/thememanager.cpp
)

set(HEADERS
    src/mainwindow.h
    src/scanner.h
    src/holycheck.h
    src/minecraftchecker.h
    src/programs.h
    src/patterns.h
    src/logs.h
    src/cacherepository.h
    src/fileinspector.h
    src/localemanager.h
    src/thememanager.h
)

qt_add_executable(FMTool ${SOURCES} ${HEADERS})
qt_add_resources(FMTool resources/resources.qrc)

target_link_libraries(FMTool PRIVATE
    Qt6::Widgets
    Qt6::Core
    Qt6::Gui
    Qt6::Svg
)

if(WIN32)
    set_target_properties(FMTool PROPERTIES WIN32_EXECUTABLE TRUE)
    target_sources(FMTool PRIVATE resources/icons/app.ico)
    set_target_properties(FMTool PROPERTIES LINK_FLAGS "-mwindows")
endif()

target_include_directories(FMTool PRIVATE ${CMAKE_CURRENT_SOURCE_DIR}/src)
```

### 5.3. Движок сканирования

**Запуск:** QtConcurrent::run или QThread с QThreadPool
**Отмена:** QAtomicInt m_cancelled (проверка в цикле)
**Обход:** QDirIterator (Files | NoDotAndDotDot, Subdirectories) + явный Stack<QString>
**Фильтр:** .jar, .exe, .dll (lowercase extension)
**Макс. размер файла:** 3000 MB (константа)

**Исключаемые директории (lowercase contains):**
```
\windows\winsxs, \windows\servicing, \windows\assembly, \windows\installer,
\windows\system32\driverstore, \windows\softwaredistribution, \windows\temp,
\windows\prefetch, \windows\csc, \$recycle.bin, \system volume information,
\programdata\package cache, \programdata\microsoft\windows\wer,
\appdata\local\microsoft\windows\inetcache, \program files, \program files (x86),
\windows, \steam, \steamlibrary, \.git, \node_modules
```

**Классификация:**
```cpp
if (CheatNames.contains(name_lower))      → "Cheat Client" (🔴)
else if (ext==".jar" && mb > 50)          → "Suspicious" (🟡)
else if (ext==".exe" && mb > 40)          → "Suspicious" (🟡)
else if (ext==".dll" && mb > 20)          → "Suspicious" (🟡)
else                                       → "Normal" (🟢)
```

**SHA-256 кеш:** SQLite (см. раздел SQLite)
- Проверка sha256 по БД → если есть и whitelisted=0 → status из кеша
- Если whitelisted=1 → пропускаем
- Если нет → вычисляем SHA-256, классифицируем, сохраняем

**OpenMP:**
```cpp
#pragma omp parallel for
```
на пакетную обработку файлов. `-fopenmp` (MinGW) / `/openmp` (MSVC)

## 18. Загрузка APPS-утилит из ресурсов

Утилиты (14 exe/jar/chm/dll/sys) находятся в папке `C:\FMTool\APPS\` на диске. При первом запуске Programs они **не встраиваются в .qrc** (слишком большие), а копируются из `APPS/` в `%TEMP%\FMTool_Apps\` при необходимости.

**Альтернатива:** добавить в resources.qrc как qt_resource (но .exe/.jar могут быть большими — до 100+ MB). Лучше: при запуске Programs проверять `%TEMP%\FMTool_Apps\` и копировать туда из `QCoreApplication::applicationDirPath() + "/../APPS/"`.

## 19. java_installer.exe — расположение

**Как используется в коде:**
```cpp
// Settings → Install Java
QString installerPath = QCoreApplication::applicationDirPath() + "/../APPS/java_installer.exe";

if (QFile::exists(installerPath)) {
```

При сборке: файл `APPS/java_installer.exe` не встраивается в QRC, просто лежит рядом с приложением.
